# Random Jokes App

Welcome to the Jokes Quotes App Repo! This Android application fetches 10 random jokes from a free API on each request. It's a simple and fun way to get a dose of inspiration, motivation, or just some random wisdom.

## Features

- Fetches 10 random jokes from a free API.
- Clean and user-friendly interface.
- Easy-to-use with swipe refresh feature for fetching new jokes.
